Bonjour,

Ce projet présente un jeu écrit en langage C.

Je l'ai crée dans le contexte de l'activité proposée dans le cours "GIT" de Marc G Gauthier sur OpenClassrooms.

Le fichier "lancement.c" correspondra à l'étape de lancement du jeu où il est demandé au joueur d'entrer son âge puis choisir son joueur.

Le fichier "jeu.c" correspondra au jeu lui même. On demande au joueur de trouver un nombre mystère compris entre 1 et 100. Pour cela une console s'ouvre et l'invite à taper un nombre. Si le nombre mystère est supérieur à celui qu'il a tapé, un message "c'est moins !" s'affiche à l'écran. Si le nombre est inférieur au nombre mystère, le message "c'est moins !" s'affiche. 
Lorsqu'il finit par trouver le nombre mystère, un message de victoire s'affiche. Le but est de le trouver en effectuant le moins de coups possibles.
